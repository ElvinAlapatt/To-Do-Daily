impoted PRIME NG component library
first `npm install primeng @primeng/themes`
then go to the app.config.ts and add the primeng to the providers

export const appConfig: ApplicationConfig = {
    providers: [
        provideAnimationsAsync(),
        providePrimeNG({
            theme: {
                preset: Aura
            }
        })
    ]
};

then you have succesfully imported primeng component library to your project

then explore all the components
